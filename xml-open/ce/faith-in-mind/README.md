# ce.faith-in-mind — 信心銘 Critical Edition

**Status:** Draft (scaffolding only — no TEI output yet)

## About

A scan-first critical edition of the 信心銘 (Faith in Mind / Xinxin Ming) attributed to the Third Patriarch Sengcan (僧璨, d. 606).

## Witnesses

8 base/collation witnesses from 4 families:

| Family | Witnesses |
|---|---|
| Standalone | 三祖大師信心銘 (Korea/Commons) |
| 四部録 | NDL 2537640, Kyoto RB00009461 (1629), Kyoto RB00012929 (1631), Waseda e1087 |
| 四部録抄 | Waseda e1089, NDL 2537799 |
| 入衆日用 | Kyoto RB00016909 |

Plus 14 control witnesses (2 translation anthologies, 11 commentaries, 1 study).
